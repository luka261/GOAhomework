ახსნა

header -> განლაგებულია: ლოგო, ნავიგაცია, ძიების ბარი. საიტის ზედა ნაწილზე .
main -> მთავარი კონტენტი, რომელიც გვერდის მთავარ მიზანი.
footer -> საიტის ქვედა ნაწილი სადაც არის საკონტაქტო ინფორმაცია და სხვა დამატებითი ინფორმაცია.
aside -> გვერდითი კონტენტი, რომელიც დაკავშირებულია მთავარ კონტენტთან.
section -> სხვადასხვა სექტორები სადაც უფრო მეტი ინფორმაცია არის მსგავს რამეზე ან მთავარ ინფოსთან არის დაკავშირებული.
nav -> ნავიგაციის ელემენტი, რომელიც შეიცავს ბმულებს სხვა გვერდებზე.

Block-level Elements -> იკავებენ მთლიან ვერტიკალურ სივრცეს იწყებენ ახალ ხაზს.
Inline Elements -> არ იკავებენ მთელ სივრცეს განლაგებულია იმავე ხაზზე და არ იწყებენ ახალ ხაზს.
style attribute -> სწრაფად და მარტივად შეცვალოთ ელემენტის ვიზუალი
div -> იგი არ აქვს ვიზუალური სტილი მაგრამ ქმნის სივრცეს, რომლის საშუალებითაც შესაძლებელია სხვადასხვა ელემენტების განლაგება
